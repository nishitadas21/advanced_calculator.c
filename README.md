# Advanced Calculator in C

An interactive C-based command-line calculator. It supports:
- Addition, Subtraction, Multiplication, Division  
- Modulus (integers only), Power (x^y)  
- Square root operation, robust error handling  
- Continuous looping interface  
- File-based operation logging to `history.txt`
